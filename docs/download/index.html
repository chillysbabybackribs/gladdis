What this build is

This page is wired for Linux-first distribution while the final product naming, brand assets, and broader platform packaging are still being finished.

Formats

AppImage works well for a portable install. The Debian package is better when you want the app registered with your system launcher.

Publishing flow

Tagged releases publish Linux artifacts to GitHub Releases, and this page automatically points to the latest published build.

Verify and install

Verify the checksum before installing if you want an integrity check from the published release bundle. 

sha256sum -c SHA256SUMS.txt --ignore-missing
chmod +x Gladys-*.AppImage && ./Gladys-*.AppImage
sudo apt install ./Gladys-*.deb
